Docker

What is Docker?

  • automates deployment in software containers
  • OS level virtualization on Linux
  • uses cgroups kernel namespaces
  • REST

Software Containers

  • isolated user space instances
  • processes, disk, network, userid, CPU
  • standard deploy
  • transport everywhere
  • don’t care about content

Boot2Docker

  • VirtualBox Tiny Core Linux
  • Comes with DockerHost
  • For MacOS and Windows
  • Kitematic in MacOS

            .addPackages(true, "org.superbiz")
            .addAsManifestResource(EmptyAsset.INSTANCE, "beans.xml");

  @Inject PaymentGateway payment; (3)

Arquillian Cube

What is Arquillian Cube

  • manages lifecycle of docker containers
  • uses Arquillian remote adapters
  • orchestrate containers
  • containerless (Netty, NodeJs, Spring Boot)
  • ready for boot2docker
